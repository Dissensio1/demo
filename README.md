## Student Time Management App
Software for managing student deadlines and tracking time spent on academic tasks. Developed as part of a university curriculum. The application allows you to monitor deadline progress, estimate completion risk, and generate analytical reports.

## Technologies used
Java 17

Spring Boot 3.5.5 (+Spring Security)

MVC Architecture

PostgreSQL for DB

Maven

Additional libraries (JXLS, Telegram Bots API)

## Main functions
Full-featured interface for managing student deadlines and time entries

Tracking time spent on tasks with automatic risk assessment based on required effort norms

Generating reports on student deadlines and progress

Uploading tabular files for bulk creation of students and deadlines
